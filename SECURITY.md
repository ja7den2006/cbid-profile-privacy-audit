# Security Policy

## Responsible Use

This repository is for privacy education and responsible disclosure only.

Do not use this project for:

- Bulk username enumeration
- Email-to-wallet matching
- Coinbase account-existence checking
- Anti-bot bypasses
- Phishing or social engineering
- Publishing third-party wallet identity links

## Reporting

If you believe you found a security or privacy issue related to this research, report it to the affected vendor through their official security or bug bounty channel.

When reporting, include:

- A concise summary
- Exact endpoint and request method
- Redacted sample response
- Privacy impact
- Abuse scenarios at a high level
- Suggested mitigations

Do not include real third-party victim data.
